# PDF Multi Uploader & Preview (ตัวอย่าง)

เว็บแอปตัวอย่างสำหรับอัพโหลดหลายไฟล์ PDF, ดูพรีวิว, ตั้งชื่อไฟล์, ตรวจชื่อซ้ำ และดาวน์โหลดไฟล์ทั้งหมดเป็น ZIP เมื่อทุกไฟล์มีชื่อและไม่มีชื่อซ้ำ

วิธีใช้งาน (local):

1. เปิดไฟล์ `index.html` ในเบราว์เซอร์ (เพียงเปิดไฟล์ด้วยฟังก์ชันเปิดไฟล์ของเบราว์เซอร์)
2. กด `Choose Files` เพื่อเลือกหลายไฟล์ PDF
3. คลิก `Open` ที่การ์ดเพื่อขยายดูเต็มจอ
4. ป้อนชื่อไฟล์ในช่อง `ชื่อไฟล์` (ห้ามว่าง)
   - กด `Tab` เพื่อไปไฟล์ถัดไป
   - กด `Ctrl+Tab` เพื่อไฟล์ก่อนหน้า
   - เมื่อตั้งชื่อซ้ำ ระบบจะแจ้งเตือนและขีดเส้นไฟล์ที่ซ้ำ
5. เมื่อทุกไฟล์มีชื่อและไม่มีชื่อซ้ำ ปุ่ม `Download All` จะเปิดใช้งาน สามารถดาวน์โหลดเป็น `pdfs.zip`

หมายเหตุ:
- ตัวอย่างนี้ใช้ PDF.js ในการเรนเดอร์พรีวิว และใช้ JSZip + FileSaver สำหรับรวมและดาวน์โหลดไฟล์เป็น ZIP
- หากต้องการโฮสต์ โปรดวางไฟล์ทั้งหมดในโฟลเดอร์เดียวกันบนเซิร์ฟเวอร์หรือเปิด `index.html` ในเบราว์เซอร์
